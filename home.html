Installation

Setup Instructions

This only works if you are on computer. If you arent on computer then this wont work.

1. Get the plugin simply by clicking the 'Hazer Plugin' button.

Hazer Plugin


2. Once you are on ROBLOX click Install then it will bring you to ROBLOX Studio.

3. Go into where you would like to use the plugin.

4. Click the plugin icon then enjoy!
